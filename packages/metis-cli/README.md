# @mindlynx/metis-cli

`npx @mindlynx/metis-cli` - init scaffolds a project, `up`/`dev` runs the collapsed single-process runtime (core + orchestrator + worker) and manages a local Temporal dev server for you. The fastest path from zero to a running workflow.

See [docs/architecture.md](../../docs/architecture.md) for how the packages fit together.
